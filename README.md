EpiTrack: Global Epidemic Intelligence Dashboard



CodeCure Biohackathon Track C: Epidemic Spread Modeling



EpiTrack is an end-to-end epidemic surveillance and forecasting platform. Built using the Johns Hopkins University (JHU) CSSE COVID-19 dataset, this project bridges the gap between raw epidemiological data and actionable biological insights. It features automated data preprocessing, a Bayesian time-series forecasting model (Meta Prophet), and an interactive Streamlit dashboard for real-time risk assessment.



\---



\##  Methodology \& Biological Insights

A key objective of this project is to translate computational methods into biologically meaningful interpretations. Our technical choices were driven by real-world epidemiological dynamics:



\* \*\*Signal Smoothing (7-Day Rolling Average):\*\* Raw daily case counts are plagued by reporting artifacts (e.g., weekend testing dips followed by Monday spikes). By applying a 7-day rolling average, we strip away these administrative anomalies to reveal the true biological transmission trend of the virus.

\* \*\*Non-Linear Wave Modeling (Meta Prophet):\*\*

&#x20; Viral outbreaks do not follow simple linear trajectories; they occur in distinct "waves" driven by viral mutations, changing seasonality, and public health interventions. We selected Prophet over traditional ARIMA because its Bayesian changepoint detection is mathematically designed to adapt to these sudden structural shifts in the infection rate. Furthermore, its multiplicative seasonality perfectly captures how viral spread scales proportionally with the size of the infected population.

\* \*\*Risk Mapping (Log₁₀ Scale \& Data-Driven Tiers):\*\*

&#x20; Because viral spread is exponential, raw case counts span multiple orders of magnitude. A linear color scale would visually wash out emerging hotspots. We applied a Log₁₀ transformation to the risk map to ensure early-stage exponential growth in smaller regions is visible alongside massive global outbreaks. Risk tiers (Low, Moderate, High, Critical) were established using strict statistical quantiles (25th, 50th, 75th percentiles) rather than arbitrary cutoffs.
